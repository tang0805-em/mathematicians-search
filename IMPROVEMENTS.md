### 2. 节点大小动态化 📊
**特性**：节点大小现在根据数学家的重要性（连接数）动态变化

**计算公式**：
```javascript
节点大小 = Math.sqrt(连接数) × 3 + 基础大小
- 中心节点基础大小：18px，最大约36px
- 周边节点基础大小：10px，最大约22px
```

**例子**：
- 欧拉（4个连接）:    ~21px
- 具有全5个学生的：   ~25px
- 孤立节点（0个连接）: ~10px

### 3. 关系标签显示 🏷️
**特性**：现在在图中的边上显示关系类型

**标签类型**：
- **师承** (mentor) → 老师-学生关系
- **合作** (collaboration) → 共同研究者
- **同事** (colleague) → 同事关系

**位置**：标签显示在两个节点的连接线中点

### 4. 可视化改进 🎨
- 节点添加阴影效果，增加深度感
- 节点文本改为白色，对比度更高
- 边界标签采用灰色显示，避免遮挡关系线
- 改进了碰撞检测，节点间距更合理

## 🚀 如何使用

### 启动应用
```bash
# 后端启动
cd backend
python app.py

# 前端启动（新终端）
cd frontend
python -m http.server 8000
```

### 浏览器访问
打开 http://localhost:8000

### 搜索新数学家
直接在搜索框中输入：
- 中文名：阿贝尔、伽罗瓦、陈景润、席南华
- 英文名：Abel、Galois、Chen Jingrun、Xi Nanhua

### 观察视觉变化
- 搜索"欧拉"：中心节点很大（多个学生连接）
- 搜索"阿贝尔"：中等大小节点（单个老师连接）
- 搜索"高斯"：非常大的节点（多位学生）
